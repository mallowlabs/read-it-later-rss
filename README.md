Read It Later RSS
====================================
Pocket (Read It Later) RSS Generator.

Deploy commands
---------------

Push

   $ git push origin master

Migrate

   $ heroku rake db:migrate

Cron Setup

   $ heroku addons:add scheduler

Cron Test

   $ heroku rake cron

Cron Log

   $ heroku logs:cron
